Documentation

Core class of ORCO.

It manages database with results and starts computations

For SQLite:

>>> runtime = Runtime("sqlite:///path/to/dbfile.db")

For Postgress:

>>> runtime = Runtime("postgresql://<USERNAME>:<PASSWORD>@<HOSTNAME>/<DATABASE>")
